# Sunny Scheduler

A lightweight stream schedule generator.

## Features

- Relative background colors
- Automatically advancing dates
- Automatic Discord timestamps

## Getting Started

### Installation

```bash
npm install
```

Then:

### Dev Build

```bash
npm run dev
```

Or:

### Production Build

```bash
npm run build
```

## Docker

To build and run using Docker:

```bash
docker build -t my-app .

# Run the container
docker run -p 3000:3000 my-app
```
